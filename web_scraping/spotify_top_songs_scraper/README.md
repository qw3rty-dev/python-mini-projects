# Spotify Most Streamed Songs Scraper

Scrapes the top 100 most streamed songs on spotify from Wikipedia.

## Link
https://en.wikipedia.org/wiki/List_of_Spotify_streaming_records

## Data collected

- Rank
- Song Title 
- Artist
- Streams (Billions)
- Release Date

## Tools & Libraries
- requests
- BeautifulSoup
- lxml
- csv

## How to run
- Install required libraries(pip install....)
- Run the script: python main.py
