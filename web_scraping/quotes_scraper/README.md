# Quotes Scraper

A beginner python web scraping project that extracts quotes,authors, and tags from a website.

# Link
https://quotes.toscrape.com

## Features
- Multi-page scraping
- Saves data to CSV
- Extracts quote, author, and tags

## Tools & Libraries
- requests
- BeautifulSoup
- lxml
- csv

 ## How to run
- Install required libraries(pip install....)
- Run the script: python main.py
